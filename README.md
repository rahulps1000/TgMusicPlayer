# TgMusicPlayer

## Note

Neither this, or PyTgCalls are stable.

## Requirements

- FFmpeg
- NodeJS [nodesource.com](https://nodesource.com/)
- Python 3.7+
- [PyTgCalls](https://github.com/pytgcalls/pytgcalls)

(Requirements are already instaled when you deploy in Heroku)
